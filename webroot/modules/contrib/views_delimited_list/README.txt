INTRODUCTION
------------

The View Delimited List module provides a new display style to Views:
There are a few configurable options determining exactly what delimiter to use, what to do right before the last item, etc.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/views_delimited_list

 * To submit bug reports and feature suggestions, or track changes:
   https://www.drupal.org/project/issues/views_delimited_list

INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit
   https://www.drupal.org/node/1897420 for further information.

CONFIGURATION
-------------

 * Configure the display format in Views:

   - Change "Format" to be "Delimited text list.

     Customize your style by click on "Settings" next to "Delimited text list".
     
     In order for the list items to be properly formatted inline, the fields displayed must be inline.
